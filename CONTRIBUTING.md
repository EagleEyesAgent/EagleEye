# Contributing to Eagle Eye 🦅

First off, thank you for considering contributing to Eagle Eye! It's people like you that make Eagle Eye such a great and reliable market intelligence system for the Solana ecosystem.

## Where do I go from here?

If you've noticed a bug or have a feature request, make sure to check our [Issues](../../issues) page to see if someone else has already created a ticket. If not, go ahead and [make one](../../issues/new)!

## Fork & create a branch

If this is something you think you can fix, then fork Eagle Eye and create a branch with a descriptive name.

## Get the environment running

Make sure you're running Node.js v20+. Install dependencies:

```bash
npm install
```

> **Note:** The public SDK (`@eagle-eye/sdk`) is under active development and will be released during **Phase 3 (Apex Predator)**. For now, contributions focus on the core engine, dashboard, and Telegram bot.

## Pull Request Process

1. Ensure any install or build dependencies are successfully running.
2. Update the README or relevant docs with details if you change the interface.
3. You may merge the Pull Request in once you have the sign-off of at least one core developer.
4. Keep PRs highly focused. Do not mix stylistic changes with functional updates.

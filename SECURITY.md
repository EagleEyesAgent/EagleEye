# Security Policy

## Supported Versions

Eagle Eye SDK v3.0 and latest minor releases are currently supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 3.0.x   | :white_check_mark: |
| 2.x.x   | :x:                |
| 1.x.x   | :x:                |

## Reporting a Vulnerability

**Please do NOT report security vulnerabilities through public GitHub issues.**

We will acknowledge receipt of your vulnerability report within 24 hours and strive to send you regular updates about our progress. Please provide detailed steps to reproduce the vulnerability.
